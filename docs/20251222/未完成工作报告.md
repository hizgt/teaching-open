# 未完成工作报告

## 项目概述

本项目是一个基于 Vue3 前端框架和 Go 后端框架的教学管理系统，旨在提供完整的教学资源管理和用户交互功能。项目结构包括前端 web/ 目录（Vue3 + Ant Design）、后端 api-go/ 目录（Go + Gin 框架），以及相关配置和文档。

## 已完成内容总结

- **前端开发**：已完成用户登录、课程列表、个人中心、系统管理等核心页面组件，实现了路由配置、权限管理、API 调用等基础功能。
- **后端开发**：已搭建 Go 框架基础，实现了用户认证、数据模型定义、基础 CRUD API 接口。
- **项目配置**：完成了初始化脚本、Git 提交指南、基础文档结构。
