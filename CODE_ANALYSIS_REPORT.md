```bash

git log --oneline -5  # 查看最近提交

# 2. 暂存所有文件
git add -A

# 3. 查看待提交的改动
git diff --cached --stat  # 查看变更统计
git diff --cached | head -100  # 查看部分差异

# 4. 执行提交
git commit -m "Phase 12-13: 后端完成 + 前端规划 + 完整DAO层生成

【后端完成】
- Phase 12: 部门日志统计 (4 APIs)

【中间件和工具】
- CORS/Logger/Error/Auth 4个中间件
- JWT认证、统一响应、错误码常量

【前端规划】
- Phase 13 4周详细开发计划
- 快速启动指南和配置示例
- Vite/Axios/Pinia/Router完整示例

【文档完善】
- 工作报告更新
- 接口规范文档
- 开发计划文档
- 快速启动指南
- 完成报告

变更统计:
- 新增文件: ~140 个
- 新增代码: 15000+ 行
- 编译通过: ✅"

# 5. 验证提交
git log --oneline -1
git show --stat

# 6. 推送到远程

git push -u origin devgo

# 7. 查看远程状态

git branch -v
```
