# 前后端接口报告 - 2025-12-25

> 本文档记录 Teaching Open 平台所有已实现的 API 接口，供前后端开发同步使用。

## 概要

- 维护人：开发团队
- 更新时间：2025-12-25
- API 基础路径：`/api/v1`
- 认证方式：JWT Token（Header: `X-Access-Token` 或 `Authorization: Bearer <token>`）

## 统一响应格式

```json
{
  "code": 200,
  "message": "操作成功",
  "result": {},
  "success": true,
  "timestamp": 1735084800000
}
```

## 分页响应格式

```json
{
  "code": 200,
  "result": {
    "records": [],
    "total": 100,
    "page": 1,
    "pageSize": 10
  }
}
```

## 一、认证模块

### 1.1 用户登录
- **接口**：`POST /sys/login`
- **说明**：用户名密码登录
- **授权**：无需认证
- **请求**：
```json
{
  "username": "admin",
  "password": "123456",
  "captcha": "abcd",
  "checkKey": "1234567890"
}
```
- **响应**：
```json
{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "userInfo": {
    "id": "xxx",
    "username": "admin",
    "realname": "管理员",
    "avatar": "",
    "sex": 1,
    "email": "",
    "phone": "",
    "orgCode": "",
    "status": 1,
    "userIdentity": 1,
    "roleCodes": ["admin"]
  },
  "sysAllDictItems": {}
}
```

### 1.2 手机号登录
- **接口**：`POST /sys/phoneLogin`
- **说明**：手机号验证码登录
- **授权**：无需认证
- **请求**：
```json
{
  "phone": "13800138000",
  "captcha": "123456"
}
```

### 1.3 退出登录
- **接口**：`POST /sys/logout`
- **说明**：退出登录，使 Token 失效
- **授权**：Bearer Token

### 1.4 获取验证码
- **接口**：`GET /sys/randomImage/{key}`
- **说明**：获取图形验证码
- **授权**：无需认证
- **响应**：
```json
{
  "key": "1234567890",
  "image": "data:image/png;base64,..."
}
```

### 1.5 发送短信验证码
- **接口**：`POST /sys/sms`
- **说明**：发送手机短信验证码
- **授权**：无需认证
- **请求**：
```json
{
  "phone": "13800138000",
  "smsMode": "1"
}
```

## 二、用户管理模块

### 2.1 用户列表
- **接口**：`GET /sys/user/list`
- **说明**：分页查询用户列表
- **授权**：Bearer Token
- **请求参数**：
  - `page`: 页码（默认 1）
  - `pageSize`: 每页数量（默认 10）
  - `username`: 用户名（模糊查询）
  - `realname`: 真实姓名（模糊查询）
  - `phone`: 手机号（模糊查询）
  - `status`: 状态（1-正常，2-冻结）
  - `departId`: 部门 ID
  - `roleId`: 角色 ID
- **响应**：
```json
{
  "records": [
    {
      "id": "xxx",
      "username": "admin",
      "realname": "管理员",
      "avatar": "",
      "sex": 1,
      "email": "",
      "phone": "",
      "status": 1,
      "orgCode": "",
      "workNo": "",
      "post": "",
      "createTime": "2025-01-01 00:00:00",
      "roleNames": ["管理员"],
      "departName": "总部"
    }
  ],
  "total": 100,
  "page": 1,
  "pageSize": 10
}
```

### 2.2 添加用户
- **接口**：`POST /sys/user/add`
- **说明**：创建新用户
- **授权**：Bearer Token
- **请求**：
```json
{
  "username": "newuser",
  "password": "123456",
  "realname": "新用户",
  "email": "user@example.com",
  "phone": "13800138000",
  "sex": 1,
  "status": 1,
  "workNo": "",
  "post": "",
  "school": "",
  "departIds": ["dept1", "dept2"],
  "roleIds": ["role1"]
}
```

### 2.3 编辑用户
- **接口**：`PUT /sys/user/edit`
- **说明**：更新用户信息
- **授权**：Bearer Token
- **请求**：
```json
{
  "id": "xxx",
  "realname": "更新姓名",
  "email": "",
  "phone": "",
  "sex": 1,
  "status": 1,
  "workNo": "",
  "post": "",
  "school": "",
  "departIds": [],
  "roleIds": []
}
```

### 2.4 删除用户
- **接口**：`DELETE /sys/user/delete`
- **说明**：逻辑删除用户
- **授权**：Bearer Token
- **请求参数**：`id` - 用户 ID

### 2.5 批量删除用户
- **接口**：`DELETE /sys/user/deleteBatch`
- **说明**：批量逻辑删除用户
- **授权**：Bearer Token
- **请求参数**：`ids` - 用户 ID 列表（逗号分隔）

### 2.6 重置密码
- **接口**：`PUT /sys/user/resetPassword`
- **说明**：重置用户密码
- **授权**：Bearer Token
- **请求**：
```json
{
  "username": "admin",
  "oldPwd": "123456",
  "newPwd": "654321"
}
```

### 2.7 查询用户详情
- **接口**：`GET /sys/user/queryById`
- **说明**：根据 ID 查询用户详情
- **授权**：Bearer Token
- **请求参数**：`id` - 用户 ID
