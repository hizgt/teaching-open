## 一、项目概述

Teaching Open 是一个面向教育机构的在线教学管理平台，当前正在进行从 Spring Boot + Vue2 到 GoFrame V2 + Vue3 的技术架构迁移。

### 迁移目标
- **后端**: Java Spring Boot → Golang GoFrame V2
- **前端**: Vue2 + Webpack → Vue3 + Vite + TypeScript
- **预期收益**: 性能提升50%+，资源占用降低70%，部署简化
